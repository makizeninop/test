# Katakana Drill App

This repository contains a minimal [Astro](https://astro.build/) project for practicing Japanese katakana characters.

## Usage

1. Install dependencies (requires internet access):

   ```bash
   npm install
   ```

2. Start the development server:

   ```bash
   npm run dev --workspace=katakana-drill
   ```

3. Open `http://localhost:4321` in your browser.

The main page lets you choose how many random katakana cards to display.
